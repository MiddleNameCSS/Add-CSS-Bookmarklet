Add CSS Bookmarklet*

* a pretty fitting name for something, I think.

Drag this to your bookmarks toolbar:
addCSS

Click it on any page to inject arbitrary CSS. Click it again or press escape while focused to hide. Changes are saved to localStorage, per domain, browser able.
